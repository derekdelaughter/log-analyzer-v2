# Security Log Analyzer CLI

A python-based security log analysis tool designed to detect suspicious
activity such as failed logins, brute-force attempts,
and abnormal IP behavior.

##
- Recursive log scanning
- Keyword and regex search
- failed login detection
- Brute-force IP detection
- IP frequency analysis
- JSON security reports
- interactive menu + CLI flags
